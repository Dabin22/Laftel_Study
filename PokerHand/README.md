#3. Poker hand -> 5개의 카드 표기로 이루어진 문자열 두 개를 받아서, 승/패/무승부를 판별하는 문제입니다.

##풀이과정
 > 1. 문자열에서 카드의 값과 종류를 각각 분류해서 저장합니다.(저장하는 방식은 카드의 값을 배열의 인덱스로 사용하여 그 인덱스의 값을 1씩 증가시키는 방식을 사용했습니다.)
 > 2. 값과 종류에 따른 계급을 정합니다.(값을 모아둔 배열의 값으로 페어나 스트레이트 여부를 계산하고 각 인덱스 값을 이용하여 동점일시 비교하는 최대값으로 사용합니다. 종류를 모아둔 배열에서는 플러쉬 여부를 계산합니다.)  
 > 3. 비교합니다. (계급을 비교해서 차이가 있으면 결과 출력 같으면 카드의 높은 값을 비교해서 결과를 출력합니다.)
